# Linux cheat sheet

| Command             | Description                                       |
| ------------------- | ------------------------------------------------- |
| lscpu               | Display some CPU information                      |
| lsblk               | Get an overview of your block devices at a glance |
| free -m             | Show RAM information                              |
| dmidecode -t bios   | Display BIOS information                          |
| dmidecode -t system | Show system information                           |

## Misc

| Command | Description              |
